FCS Teaching Topics: Brightspace Tips

TAKE-AWAYS

  • Using an LMS (Learning Management System) is convenient and saves you time by providing access to course content, auto marked quizzing features and discussion forums
  • There is a Brightspace Learning Community where you can find video tutorials, "how-tos", ask questions and get advice: https://community.d2l.com/brightspace/
  • There are Sample Module Outlines Available as Downloads (for Higher Education): https://community.desire2learn.com/d2l/home/13796
  • Supportive environment - develop a collegial, kind and approachable relationship between you individually and among your TAs and Markers as a group. This will assist in developing a support network so the onus isn't always on you! Saving you time!

Why use Brightspace?

Brightspace is the LMS (Learning Management System) that Dalhousie uses to create online learning environments for learning and teaching. Most courses that are created on Brightspace are for blended learning - whereby resources and some of the functionality of the online platform are utilized, but students still attend face-to-face lectures, instruction and labs. However, Brightspace can also be used as a robust platform for courses that are offered purely online.
  • organizes course content in one location (documents, videos and various files etc), anytime/anywhere access
  • easily tracks students' progress and performance (digital grade book, assignments, quizzes etc)
  • automatically graded quizzes (multiple choice)
  • online discussion posts, forums, notices
  • Collaborate Ultra (video conferencing/recording whereby students can access synchronously (live) or asynchronously (prerecorded)
  • Courses can be copied - so they can be utilized in a future offering - saving you time!

When learning how to use Brightspace, there is so much to learn that it may seem overwhelming, so it's important to take it step by step. Focus on the first thing that you have to do, for example, post course content, then learn how to create the Grade Book, create Assignments, Quizzes etc. The video tutorials above very useful for getting started.
